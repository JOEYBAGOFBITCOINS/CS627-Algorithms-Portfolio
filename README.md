CS627-Algorithms-Portfolio

This repository contains implementations of several algorithm design paradigms studied throughout the course.

Unit 1  Algorithm Analysis
Implementation of an in-place array reversal algorithm demonstrating time complexity O(n) and constant space complexity.

Unit 2  Divide and Conquer
Implementation of a ternary search algorithm used to locate elements in a sorted dataset by dividing the search space into three segments.

Unit 3  Greedy Algorithms
Implementation of a greedy disk allocation strategy similar to the Best-Fit Decreasing approach used in bin-packing problems.

Unit 4  Dynamic Programming
Implementation of an image comparison algorithm that compares two black-and-white images represented as binary matrices.  
The algorithm uses dynamic programming to compute the minimal mismatch between rows using a distance matrix.
